# Proboi Roles

Открытые **роли** для AI-ассистентов. Роль — это персона + база знаний (инструкции), которые ассистент «надевает» и после этого **целиком работает как этот специалист**. Не отдельный бот, не плагин «для всех» — подключает только тот, кто захочет.

Роли универсальны: сделаны под Telegram-бот [Proboi](https://t.me/proboiAI_bot), но работают в **любом ассистенте** (Claude Code, ChatGPT и др.) — файлы из `knowledge/` это обычные текстовые инструкции.

## Как подключить роль

- **Proboi-бот** ([@proboiAI_bot](https://t.me/proboiAI_bot)): кинь ему в личке ссылку на папку роли — наденет её прямо в чате.
- **Claude Code / другой ассистент**: отдай файлы из `knowledge/` как системный промпт (или контекст).

## Что внутри роли

```
<role-name>/
├── <role-name>.role.json   — манифест: persona, задача, инструменты, эскалация (для Proboi; другие платформы игнорируют)
├── knowledge/              — база знаний: вся рабочая логика и инструкции (это и есть универсальное ядро)
└── README.md              — что делает и как подключить
```

Инструменты в `knowledge/` описаны через **возможности** («пришли файлом, если умеешь», «посчитай», «если умеешь расписание — предложи напоминание»), поэтому роль не привязана к одной платформе и мягко деградирует там, где чего-то нет.

## Добавить свою роль

Скопируй структуру существующей роли, опиши `*.role.json` и положи инструкции в `knowledge/`. Минимум кода — максимум понятных инструкций в базе знаний.

## Лицензия

MIT. Отдельные роли используют сторонние промпты как **ориентир** (без копирования текста) — атрибуция указана в README каждой такой роли.
