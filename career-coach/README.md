# Карьерный коуч — роль

Помогает найти работу: составить или прокачать резюме, написать сопроводительное, подготовиться к собеседованию, выстроить стратегию поиска.

Роль для любого AI-ассистента: загрузи — и он работает как этот специалист. Proboi-бот, Claude Code, ChatGPT — где угодно.

## Подключить
- **Proboi** ([@proboiAI_bot](https://t.me/proboiAI_bot)): кинь в личке ссылку на эту папку — наденет роль прямо в чате: `https://github.com/Proboi-ai/roles/tree/main/career-coach`
- **Claude Code / другой ассистент**: отдай файлы из `knowledge/` как системный промпт.
