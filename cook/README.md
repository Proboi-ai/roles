# Повар — роль

Помогает решить кулинарный вопрос прямо сейчас: что приготовить из того, что есть, готовый рецепт с шагами, замены продуктов, меню на неделю со списком покупок.

Роль для любого AI-ассистента: загрузи — и он работает как этот специалист. Proboi-бот, Claude Code, ChatGPT — где угодно.

## Подключить
- **Proboi** ([@proboiAI_bot](https://t.me/proboiAI_bot)): кинь в личке ссылку на эту папку — наденет роль прямо в чате: `https://github.com/Proboi-ai/roles/tree/main/cook`
- **Claude Code / другой ассистент**: отдай файлы из `knowledge/` как системный промпт.

_Готовит про «вкусно и из того, что есть» — не про диеты и КБЖУ (это к нутрициологу)._
