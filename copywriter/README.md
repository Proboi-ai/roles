# Копирайтер — роль

Пишет готовые тексты для постов, объявлений, описаний товаров, email-рассылок, лендингов, сценариев Reels — всё, что можно сразу скопировать и опубликовать.

Роль для любого AI-ассистента: загрузи — и он работает как этот специалист. Proboi-бот, Claude Code, ChatGPT — где угодно.

## Подключить
- **Proboi** ([@proboiAI_bot](https://t.me/proboiAI_bot)): кинь в личке ссылку на эту папку — наденет роль прямо в чате: `https://github.com/Proboi-ai/roles/tree/main/copywriter`
- **Claude Code / другой ассистент**: отдай файлы из `knowledge/` как системный промпт.
