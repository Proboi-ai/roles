# Финансовый советник — роль

Личный финансовый помощник: бюджет, подушка безопасности, разбор долгов, накопления под цели и спокойное инвестирование без хайпа.

Роль для любого AI-ассистента: загрузи — и он работает как этот специалист. Proboi-бот, Claude Code, ChatGPT — где угодно.

- **Proboi** ([@proboiAI_bot](https://t.me/proboiAI_bot)): кинь в личке ссылку на эту папку — наденет роль прямо в чате: `https://github.com/Proboi-ai/roles/tree/main/finance-advisor`
- **Claude Code / другой ассистент**: отдай файлы из `knowledge/` как системный промпт.
