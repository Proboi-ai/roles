# Фитнес-тренер — роль для Telegram-бота Proboi

Это **роль**, а не отдельный бот. Бот Proboi (chat-agent или клон-бот) «надевает» эту роль и полностью работает как персональный фитнес-тренер: онбординг, программа тренировок, прогрессия, ведение тренировки, прогресс-страница.

Ничего не ставится всем пользователям. Роль подключает только тот, кто захочет свой фитнес-бот.

## Что внутри

| Файл | Что это |
|---|---|
| `fitness-coach.role.json` | Определение роли: персона, задача, инструменты, триггеры, эскалация, список файлов знаний |
| `knowledge/coaching.md` | Тренерский плейбук: активация, онбординг, КБЖУ, программа, прогрессия, ведение тренировки (план списком → гифки → отдых → итоговая таблица), безопасность |
| `knowledge/progress.md` | Лог тренировок и автозаполнение, калории, прогресс-страница со стриками |
| `knowledge/quotes.md` | Багаж цитат для отдыха между подходами |
| `data/exercises.json` | База из 1094 упражнений с русскими названиями и `gif_url` (CDN ExerciseDB) |

База упражнений **в комплекте роли** (`data/exercises.json`). При первой активации тренер один раз кладёт её на диск (`fitness/exercises.json`) одной командой `curl` **через песочницу с интернетом** (`mcp__container__Bash`, не хостовый shell — у того нет сети) и берёт `gif_url` оттуда. Гифку техники присылает сразу, без веб-поиска. Если сетевая команда отвечает «requires approval» — значит использован не тот shell; правильный — `mcp__container__Bash`.

## Как загрузить

Просто пришли боту Proboi в личке ссылку на эту папку:

`https://github.com/Proboi-ai/roles/tree/main/fitness-coach`

Бот прочитает роль, скачает базу упражнений и **сразу начнёт работать тренером прямо в этом чате** — перейдёт к онбордингу, ничего не спрашивая про «куда ставить». Отдельный клон-бот или группа не нужны. Хочешь вернуть обычного ассистента — напиши боту об этом.

## Инструменты роли

- `send_file` — отправка GIF-демонстраций упражнений и графиков прогресса.
- `container` — генерация графиков (matplotlib) и сборка HTML-страницы прогресса.
- `schedule` — напоминания о тренировках (таймер отдыха не используется: пользователь ставит таймер у себя в телефоне).

## Статус

v1, обкатка механизма ролей. Тренерская логика отражает правила оригинального бота FitCoach (КБЖУ по Миффлину, линейная прогрессия +2.5/+1.25 кг, деload после 2 неудач, 1RM по Эпли).

**Про отдых между подходами:** живого посекундного таймера в чате нет, и пинги-напоминания убраны как мешающие. Вместо этого тренер просит поставить таймер у себя в телефоне, шлёт цитату и продолжает, как пользователь напишет, что готов. Настоящий тикающий таймер с кнопками (как в оригинальном FitCoach) требует доработки самого бота, не роли.
