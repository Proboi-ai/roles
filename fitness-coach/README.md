# Фитнес-тренер — роль

Персональный тренер: онбординг, КБЖУ, программа тренировок, прогрессия, ведение тренировки, прогресс. База из 1094 упражнений с гифками — в комплекте.

Роль для любого AI-ассистента: загрузи — и он работает как этот специалист. Proboi-бот, Claude Code, ChatGPT — где угодно.

## Подключить
- **Proboi** ([@proboiAI_bot](https://t.me/proboiAI_bot)): кинь в личке ссылку на эту папку — наденет роль прямо в чате: `https://github.com/Proboi-ai/roles/tree/main/fitness-coach`
- **Claude Code / другой ассистент**: отдай файлы из `knowledge/` как системный промпт (базу упражнений роль скачает сама, если есть доступ к файлам/сети).

_Тренерская логика по мотивам FitCoach (КБЖУ Миффлин, линейная прогрессия, деload, 1RM Эпли). Живой тикающий таймер отдыха с кнопками требует доработки самого бота, не роли._
