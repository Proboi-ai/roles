# Фитнес-тренер — роль

Персональный тренер: онбординг, КБЖУ, программа тренировок, прогрессия, ведение тренировки, прогресс. Готовый набор основных упражнений с гифками-демонстрациями — прямо в знаниях роли (`knowledge/exercises.md`), скачивать ничего не нужно.

Роль для любого AI-ассистента: загрузи — и он работает как этот специалист. Proboi-бот, Claude Code, ChatGPT — где угодно.

## Подключить
- **Proboi** ([@proboiAI_bot](https://t.me/proboiAI_bot)): кинь в личке ссылку на эту папку — наденет роль прямо в чате: `https://github.com/Proboi-ai/roles/tree/main/fitness-coach`
- **Claude Code / другой ассистент**: отдай файлы из `knowledge/` как системный промпт. База упражнений с гифками уже внутри (`knowledge/exercises.md`).

> `data/exercises.json` — это полный датасет-источник (1094 упражнения), из него собран компактный `knowledge/exercises.md`. Роль использует именно `exercises.md` и **ничего не скачивает** в рантайме (на бесплатном тарифе у песочницы нет сети — скачивание базы там в принципе невозможно).

_Тренерская логика по мотивам FitCoach (КБЖУ Миффлин, линейная прогрессия, деload, 1RM Эпли). Живой тикающий таймер отдыха с кнопками требует доработки самого бота, не роли._
