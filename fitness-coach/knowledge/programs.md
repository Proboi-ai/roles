# Готовые программы тренировок

Библиотека архетипов-пресетов. Когда подбираешь программу (§3 плейбука `coaching.md`) — бери архетип **отсюда** и наполняй его стандартными упражнениями этого типа (ты их знаешь). Все программы — общеизвестные методики (StrongLifts 5×5, Push/Pull/Legs, верх/низ, Strong Curves и т.п.). Подходы/повторы и прогрессию бери из §4 `coaching.md`, если у пресета нет своей. Технику давай текстом; гифки — только по запросу (§3).

## Как выбрать программу (по шагам)

После онбординга (§1 `coaching.md`) у тебя есть: пол, цель, место (дом/улица/зал), инвентарь, дней в неделю, уровень. Выбирай так:

1. **Место + инвентарь:** дом без инвентаря / улица → **Д1** или **Д3**; дом с гантелями/резинками → **Д2**; зал → **З1/З2/З3** и женский **Ж1**.
2. **Уровень:** новичок (<6 мес) → фуллбоди (**Д1, Д2, З1, Ж1**); средний (>6 мес, базу освоил) → сплиты (**З2** PPL, **З3** верх/низ).
3. **Цель:** похудение → любая силовая + дефицит калорий (§2) + по желанию круговая **Д3** (не заменяй силовые одним кардио); масса/сила/рекомпозиция → силовая по месту и уровню.
4. **Дней:** 2–3 → фуллбоди (**Д1, Д2, З1, Ж1**); 4 → верх/низ (**З3**); 5–6 → PPL (**З2**).
5. **Пол** задаёт упор (раздел «Упор по полу»), упражнения те же. **Названная цель важнее пола.**

Мало данных — бери простейший подходящий по месту фуллбоди и уточняй по ходу.

## Пресеты (архетипы)

Прогрессия по умолчанию — линейная, §4 `coaching.md` (+2.5 кг базовые / +1.25 изоляция / +1–2 повтора для веса тела; 2 провала подряд → деload −10%). Где у программы своя — указано.

- **Д1 — дом/улица, вес тела, фуллбоди (новичок).** 3×/нед через день, 3 круга: присед, отжимания (регресс с колен / прогресс ноги выше), обратные отжимания на трицепс, австралийские подтягивания, выпады, планка. Прогрессия: взял верх диапазона — усложни вариант или +повторы.
- **Д2 — дом, гантели/резинки, фуллбоди (новичок–средний).** 3×/нед: гоблет-присед, жим гантелей лёжа, тяга гантели в наклоне, румынская тяга, жим гантелей стоя, бицепс/трицепс, планка.
- **Д3 — жиросжигание, круговая (дом/зал).** 3×/нед, 3 круга подряд высоким темпом, отдых между кругами 1–2 мин: присед, отжимания, выпады, тяга одной рукой, планка, jumping jacks. Прогрессия: сперва чистая техника → +повторы → сокращай отдых.
- **З1 — зал, штанга, фуллбоди 3× (новичок), StrongLifts 5×5.** Чередуй A/B через день. A: присед 5×5, жим лёжа 5×5, тяга штанги в наклоне 5×5. B: присед 5×5, жим стоя 5×5, становая 1×5. Старт с пустого грифа, +2.5 кг/тренировку (присед и становая +5); провал 5 повторов → повтори вес; 3 провала → −10%.
- **З2 — зал, Push/Pull/Legs (гипертрофия, средний).** 3 или 6 дней. Push: жим лёжа, жим стоя, жим на наклонной, разгибания на трицепс, махи в стороны. Pull: становая/тяга штанги в наклоне, верхний блок, тяга сидя, фейспулл, бицепс. Legs: присед, румынская тяга, жим ногами, сгибание ног, икры. Базовое первым (6–10), изоляция 8–15.
- **З3 — зал, верх/низ 4× (средний).** Верх: жим лёжа, тяга штанги в наклоне, жим на наклонной, верхний блок, жим стоя, бицепс, трицепс. Низ: присед, румынская/становая тяга, жим ногами, сгибание ног, икры, планка.
- **Ж1 — зал, упор ягодицы/низ (Strong Curves).** 3 дня A/B/C через день, ягодичное упражнение всегда первым. A: ягодичный мост, тяга гантели, болгарский присед, жим гантелей лёжа, румынская тяга, отведение бедра, планка. B: мост на одной ноге, верхний блок, зашагивания, жим стоя, гиперэкстензия, «ракушка» с резинкой, боковая планка. C: мост-марш (статика), тяга сидя, присед с гантелями, жим на наклонной, румынская на одной ноге, ходьба с резинкой. Фазово: нед 1–4 вес тела/гантели → 5–8 переход на штангу (мост → хип-траст).

## Упор по полу (НЕ запрет упражнений)

Упражнения для всех **одни и те же** — «не женских» упражнений не существует, запрещать ничего не нужно. Пол меняет три вещи:

1. **Упор по умолчанию** (если цель не названа явно): девушка → больше объёма на ягодицы/ноги/заднюю цепь и кор (мосты, хип-траст, румынская, выпады, зашагивания), верх есть, но меньше подходов и чаще гантели/тренажёры; база — **Ж1** или любой пресет + 1–2 ягодичных в начало. Парень → больше объёма на верх (жимы, тяги, подтягивания) + ноги; база — **З1/З2/З3**.
2. **Стартовые веса:** у женщин в среднем слабее верх (≈50% от мужского) при близком к мужскому низе (≈70%) — это норма. Жимы/тяги начинай с гантелей/пустого грифа/резинки, низ можно грузить смелее. Прогрессируй каждое движение отдельно, не «дотягивай» верх до низа.
3. **Формулировки:** девушке ближе «форма, тонус», парню «масса, сила» — это лишь ярлыки, веди по реальной цели.

**Железное правило: названная цель важнее пола.** Девушка хочет жать и тянуть — даёшь силовой (**З1/З2**), без «только ягодицы». Парень хочет похудеть — даёшь жиросжигание (**Д3** + дефицит), без «качалки на массу».

**Не повторяй мифы:** тяжёлые веса НЕ делают девушку «перекачанной» (тестостерон в 15–20 раз ниже); «тонус» — это мышцы при низком проценте жира, отдельных «тонизирующих» упражнений нет; кардио не эффективнее силовых для похудения.

## Женская физиология (безопасность)

- **Цикл:** строить программу по фазам цикла научно не оправдано. Правильно — разрешить снижать интенсивность в дни плохого самочувствия по ощущениям, без привязки к календарю.
- **Беременность:** силовые при нормальной беременности полезны (ACOG), но со 2 триместра убери упражнения лёжа на спине (замени на наклонную/стоя), без прыжков и ударных, без задержки дыхания под нагрузкой, без риска падений. Только с одобрения её врача.
- **Послеродовой период:** первые ~6 недель — восстановление тазового дна (дыхание, Кегель), без прямых скручиваний и жёсткой планки до закрытия диастаза. Полноценные силовые обычно с 12–16 недели (после кесарева — позже).
- Любые сомнения по женскому здоровью → **эскалируй к врачу** (§7 `coaching.md`). Ты не врач.
