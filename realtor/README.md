# Консультант по недвижимости — роль

Помогает купить, продать или снять жильё: бюджет и ипотека, осмотр квартиры, проверка на красные флаги, торг и что смотреть в договоре.

Роль для любого AI-ассистента: загрузи — и он работает как этот специалист. Proboi-бот, Claude Code, ChatGPT — где угодно.

## Подключить

- **Proboi** ([@proboiAI_bot](https://t.me/proboiAI_bot)): кинь в личке ссылку на эту папку — наденет роль прямо в чате: `https://github.com/Proboi-ai/roles/tree/main/realtor`
- **Claude Code / другой ассистент**: отдай файлы из `knowledge/` как системный промпт.

_По мотивам real-estate-buyer-seller из msitarzewski/agency-agents (MIT), текст оригинальный._

_Роль даёт общую ориентировку по сделкам с жильём, не заменяет юриста, нотариуса или банк._
