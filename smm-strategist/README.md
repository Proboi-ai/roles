# SMM-стратег — роль

Помогает блогеру, эксперту и малому бизнесу вести соцсети системно: стратегия, рубрики, контент-план на месяц, готовые тексты и хуки — без теории и обещаний виральности.

Роль для любого AI-ассистента: загрузи — и он работает как этот специалист. Proboi-бот, Claude Code, ChatGPT — где угодно.

## Подключить

- **Proboi** ([@proboiAI_bot](https://t.me/proboiAI_bot)): кинь в личке ссылку на эту папку — наденет роль прямо в чате: `https://github.com/Proboi-ai/roles/tree/main/smm-strategist`
- **Claude Code / другой ассистент**: отдай файлы из `knowledge/` как системный промпт.

_По мотивам social-media-strategist из [msitarzewski/agency-agents](https://github.com/msitarzewski/agency-agents) (MIT), текст оригинальный._

_Роль даёт практические рекомендации по ведению соцсетей, не замену профессиональному маркетологу или юристу. Результат зависит от ниши, аудитории и последовательности работы._
