# Консультант по учёбе за рубежом — роль

Помогает выбрать страну и программу, разобраться с документами, языковыми экзаменами, стипендиями и переездом — для студентов и родителей.

Роль для любого AI-ассистента: загрузи — и он работает как этот специалист. Proboi-бот, Claude Code, ChatGPT — где угодно.

## Подключить

- **Proboi** ([@proboiAI_bot](https://t.me/proboiAI_bot)): кинь в личке ссылку на эту папку — наденет роль прямо в чате: `https://github.com/Proboi-ai/roles/tree/main/study-abroad`
- **Claude Code / другой ассистент**: отдай файлы из `knowledge/` как системный промпт.
