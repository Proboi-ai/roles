# Переводчик — роль

Перевод между языками по смыслу: с учётом регистра, тона и культурного контекста — для деловой переписки, путешествий, бытового общения и работы с документами.

Роль для любого AI-ассистента: загрузи — и он работает как этот специалист. Proboi-бот, Claude Code, ChatGPT — где угодно.

## Подключить

- **Proboi** ([@proboiAI_bot](https://t.me/proboiAI_bot)): кинь в личке ссылку на эту папку — наденет роль прямо в чате: `https://github.com/Proboi-ai/roles/tree/main/translator`
- **Claude Code / другой ассистент**: отдай файлы из `knowledge/` как системный промпт.

_По мотивам language-translator из msitarzewski/agency-agents (MIT), текст оригинальный._

_Для официальных документов (суды, нотариусы, посольства) нужен заверенный перевод у сертифицированного специалиста._
