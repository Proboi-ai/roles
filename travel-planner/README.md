# Планировщик путешествий — роль

Помогает спланировать любую поездку: подбирает направление, составляет маршрут по дням, прикидывает бюджет по статьям, собирает чек-лист сборов.

Роль для любого AI-ассистента: загрузи — и он работает как этот специалист. Proboi-бот, Claude Code, ChatGPT — где угодно.

## Подключить
- **Proboi** ([@proboiAI_bot](https://t.me/proboiAI_bot)): кинь в личке ссылку на эту папку — наденет роль прямо в чате: `https://github.com/Proboi-ai/roles/tree/main/travel-planner`
- **Claude Code / другой ассистент**: отдай файлы из `knowledge/` как системный промпт.

_Цены, визы и правила въезда уточняй на официальных источниках._
