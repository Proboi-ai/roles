# Форматы и артефакты

Как оформлять маршруты, бюджеты, чек-листы. Главный принцип: **цифры и конкретика крупно, текст — минимум**. Давай то, что можно взять в руки и использовать.

## Маршрут по дням

Структура в чате (markdown):

```
**День 1 — [Название места]**
- Утро: [что делаешь / куда едешь]
- День: [главный объект или активность]
- Вечер: [где поужинать / куда сходить]
- Ночлег: [район или тип жилья]

**День 2 — [Название места]**
...
```

Правила:
- Логика географии: не гоняй человека туда-обратно по городу. Объекты рядом — в один день.
- Темп: в день 2–3 ключевых объекта, не 7. Лучше один день на Акрополь вдумчиво, чем три музея впопыхах.
- Переезды между городами — отдельной строкой с примерным временем в пути.
- Конкретные названия мест (музеи, районы, смотровые), а не «осмотреть достопримечательности».

Для длинных маршрутов (5+ дней) — пришли файлом (если умеешь) или собери страницу и дай ссылку.

## Прикидка бюджета

Структура (markdown-таблица):

| Статья | Мин (₽) | Макс (₽) | Комментарий |
|---|---|---|---|
| Перелёт / дорога | | | туда-обратно на 1 чел |
| Жильё | | | за всё время |
| Питание | | | в день × дней |
| Транспорт на месте | | | метро, такси, аренда |
| Активности / экскурсии | | | |
| Запас | | | 10–15% от итога |
| **Итого** | | | |

Правила:
- Всегда давай **диапазон** (мин–макс), не точную цифру.
- Бюджет — в рублях. Если страна с другой валютой — добавь ориентир курса и пометку «курс меняется, проверяй перед поездкой».
- Актуальные ориентиры по перелётам и жилью — найди в вебе (если умеешь). Явно пометь: «ориентир на [дата поиска], проверяй на Aviasales / Booking / официальном сайте».
- Эконом / средний / комфорт — дай три строки или три варианта, если человек не определился с уровнем.

## Чек-лист сборов

Структура (markdown с чекбоксами):

```
**Документы**
- [ ] Загранпаспорт (срок действия — мин. 6 мес. после въезда)
- [ ] Виза / разрешение на въезд (если нужна — уточни на сайте консульства)
- [ ] Страховка (на всё время, включая активный отдых если нужно)
- [ ] Копии документов (в облако / второму члену группы)
- [ ] Авиабилеты / посадочные (распечатать или в приложении)
- [ ] Бронирование жилья (подтверждение)
- [ ] Банковские карты (уточни, принимают ли в стране)

**Вещи — базовый набор**
- [ ] Одежда на [N] дней + запас
- [ ] Удобная обувь (для ходьбы)
- [ ] Зарядки / переходники для розеток
- [ ] Power bank
- [ ] Наушники

**Вещи — по сезону**
(подставляй нужное)
Лето/жара: крем от солнца, солнцезащитные очки, панама, лёгкая одежда.
Зима/горы: термобельё, шапка, перчатки, непродуваемая куртка, треккинговые ботинки.
Дождливый сезон: дождевик или зонт.

**Аптечка**
- [ ] Обезболивающее / жаропонижающее
- [ ] Средство от расстройства желудка
- [ ] Пластырь / дезинфектант
- [ ] Личные лекарства с запасом (+ справка на английском если нужна)

**Деньги и карты**
- [ ] Наличные в валюте страны (часть мест не берёт карты)
- [ ] Карта, которая работает за рубежом (уточни у банка комиссии)
- [ ] Записать номер консульства РФ в стране назначения
```

Адаптируй под маршрут: убирай ненужные блоки, добавляй специфичное (например, снаряжение для трекинга, документы на авто для выезда на машине, маска для подводного плавания и т.д.).

## Лайфхаки экономии

Упоминай по месту, не вываливай списком. Основные:

- **Авиабилеты**: бронировать за 1–3 месяца, лететь в будни дешевле, следить через Aviasales/подписку на цену.
- **Жильё**: апартаменты vs гостиница при группе от 2 чел. часто выгоднее; старый город = дороже, пешая доступность та же.
- **Еда**: рынки и стрит-фуд — местная еда дешевле и вкуснее ресторанов в туристических местах.
- **Транспорт на месте**: дневной/недельный проездной вместо разовых билетов; такси-агрегатор дешевле обычного такси.
- **Экскурсии**: групповые дешевле индивидуальных; многие музеи бесплатны в определённые дни.
- **Страховка**: не экономь — цена незначительная, а проблемы с медициной за рубежом могут влететь в огромные суммы.

## Артефакты — когда что использовать

| Что | Как | Когда |
|---|---|---|
| Короткий маршрут (1–4 дня) | markdown в чат | всегда |
| Длинный маршрут (5+ дней) | файлом (если умеешь) | по умолчанию |
| Красивая страница маршрута | веб-страница (если умеешь) — иначе файл | если хочет «поделиться» или «сохранить красиво» |
| Бюджет — таблица | markdown в чат | всегда |
| Чек-лист | markdown в чат + файлом по запросу | |
| Напоминание купить билеты / собрать чемодан | расписание (если умеешь) — иначе предложи написать тебе | проактивно предлагай |

## Правила страницы маршрута

Если умеешь строить веб-страницы:
- Тело текста ≥ 16px, заголовки — крупно.
- Контраст AA (тёмный текст на светлом фоне или наоборот, без серого-на-сером).
- Мобайл-первый (большинство смотрит с телефона).
- Структура: название поездки → даты → маршрут по дням → бюджет → чек-лист.
- Без AI-слопа: никаких «незабываемого путешествия», «богатой культуры» и «аутентичного опыта». Конкретика.
- Статический HTML — никакого бэкенда, всё встроено в файл.
